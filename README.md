This repository contains the Python notebooks for simulating and determining waveform degeneracies of quasi-circular binary systems and Lagrange three-body systems up to the 0.5PN order. The three main notebooks are as follows:

- `waveforms.py` defines the waveforms of the quasi-circular binary and the Lagrange triple up to the 0.5PN order.
- `Part 1 - Theory Simulations.ipynb` contains the code for analyzing and determining the degeneracy at the mass quadrupole order and at the 0.5PN order.
- `Part 2 - Simulations with LIGO Data.ipynb` contains the code for calculating the waveform match of degenerate systems using the power spectral density of the advanced LIGO design.
